sklearn_classifier.sklearn_classifier module¶

sklearn_classifier.sklearn_classifier.train_model(context: mlrun.execution.MLClientCtx, model_pkg_class: str, dataset: mlrun.datastore.base.DataItem, label_column: str = 'labels', encode_cols: List[str] = [], sample: int = - 1, test_size: float = 0.3, train_val_split: float = 0.7, test_set_key: str = 'test_set', model_evaluator=None, models_dest: str = '', plots_dest: str = 'plots', file_ext: str = 'parquet', model_pkg_file: str = '', random_state: int = 1) → None[source]¶

train a classifier

An optional cutom model evaluator can be supplied that should have the signature: my_custom_evaluator(context, xvalid, yvalid, model) and return a dictionary of scalar “results”, a “plots” keys with a list of PlotArtifacts, and and “tables” key containing a returned list of TableArtifacts.

Parameters

  • context – the function context

  • model_pkg_class – the model to train, e.g, “sklearn.neural_networks.MLPClassifier”, or json model config

  • dataset – (“data”) name of raw data file

  • label_column – ground-truth (y) labels

  • encode_cols – dictionary of names and prefixes for columns that are to hot be encoded.

  • sample – Selects the first n rows, or select a sample starting from the first. If negative <-1, select a random sample

  • test_size – (0.05) test set size

  • train_val_split – (0.75) Once the test set has been removed the training set gets this proportion.

  • test_set_key – key of held out data in artifact store

  • model_evaluator – (None) a custom model evaluator can be specified

  • models_dest – (“”) models subfolder on artifact path

  • plots_dest – plot subfolder on artifact path

  • file_ext – (“parquet”) format for test_set_key hold out data

  • random_state –

    1. sklearn rng seed
