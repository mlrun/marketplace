Parameters

  • context – Function context.

  • dataset – Raw data file.

  • label_column – (label) Ground-truth y labels.

  • train_validation_size – (0.75) Train validation set proportion out of the full dataset.

  • sample – (1.0) Select sample from dataset (n-rows/% of total), randomzie rows as default.

  • models_dest – (models) Models subfolder on artifact path.

  • test_set_key – (test_set) Mlrun db key of held out data in artifact store.

  • plots_dest – (plots) Plot subfolder on artifact path.

  • dask_function – dask function url (db://..)

  • dask_client – dask client object

  • file_ext – (parquet) format for test_set_key hold out data

  • random_state –

    1. sklearn seed
